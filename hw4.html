My Note For HTML & CSS Lists

In HTML there are 3 different types of list. Below you will find the list name with a brief defition, and its usage.

Lists Usage
Unodered List Displays each item in bullet list.
<ul>
   <li>item 1</li>
   <li>item 2</li>
</ul>
Ordered List Displays each item in pair list, used for glossary, vocabulary, question & answers, etc.
<ol>
   <li>item 1</li>
   <li>item 2</li>
</ol>
Definition List Displays each item in number list.
<dl>
   <dt>Term 1</dt>
      <dd>Definition 1</dd>
   <dt>Term 2</dt>
      <dd>Definition 2</dd>
</dl>
